Subversion to git conversion process


1. Checkout subversion repo:
$ svn co svn+ssh://home/data/svnroot/software/daemons/ntp/ntp ntp-ORIG

NOTE: Using "ORIG" extension to alleviate git "ntp" repo name conflict.

2. Get authors for "ntp" repo in users.txt file:
$ cd ntp-ORIG
$ ../svn-to-git-user
$ mv users.txt ..
$ cd ..
$ cat users.txt 
marcilr = Raymond Marcil <marcilr@gmail.com>
$

Looks good.

3. In present working directory use "git svn clone" to convert "ntp" repo:
$ git svn clone svn+ssh://home/data/svnroot/software/daemons/ntp/ntp \
--authors-file=./users.txt --no-metadata ntp
